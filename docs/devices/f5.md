# 0xF5 — SZM / LKM2 / CSU (chassis-dependent)

**Status:** Stub.

**Role:** The same address hosts **two different devices** depending on chassis:

- **E31**: LKM2 — Lamp Control Module 2.
- **Most other chassis**: SZM — Schaltzentrum-Mittelkonsole (Centre Console Switch Centre). The cluster of switches in the centre console: hazard, climate-recirculate, sport mode, etc.

bimmerz at `devices.ts:18` labels the module **CSU** ("Centre Switch Control Unit") — the same physical device as SZM under a different mnemonic.

**Buses:** K.
