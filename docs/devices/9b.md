# 0x9B — CVM (E36) / SPMFT (E46) (chassis-dependent)

**Status:** Stub.

**Role:** The same address hosts **two different devices** depending on chassis:

- **E36**: CVM — Convertible Soft Top Module (the E36 convertible's roof controller).
- **E46**: SPMFT — Mirror Memory, Driver. (Reassignment on later chassis where the E36 CVM no longer exists.)

This is a clean example of BMW's address-pool-reuse pattern (see [`../protocol/addressing.md`](../protocol/addressing.md#categories-of-address)).

**Buses:** K.
